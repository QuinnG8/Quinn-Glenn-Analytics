NFL Draft Analytics

Wide Receiver Analysis · 2017–2025

Project Overview

This project explores the relationship between college production, combine athleticism, and NFL success for wide receivers drafted from 2017 to 2025. The goal was to identify which metrics actually predict professional performance and where teams find the best value in the draft.

Tools Used
Python Pandas SQL SQLite Matplotlib
Key Questions
  • Which college stats matter most for NFL success?
  • What's the hit rate by draft round?
  • Does athleticism predict performance?
